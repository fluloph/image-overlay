# Image Composition Tool (Overlay Studio)

A web-based tool for composing images with dynamic overlays.

## Features
- **Background Upload**: Upload a base image.
- **Dynamic Overlays**: Add multiple overlay images on top of the background.
- **Text Layer**: Add customizable text with font, color, and background opacity settings.
- **Privacy First**: All processing happens locally in your browser. No images are uploaded to a server.
- **Export**: Download your final composition as a PNG.

## Usage
1.  Open the application (hosted or local).
2.  Upload a **Background Image**.
3.  (Optional) Enable and configure the **Text Overlay**.
4.  Click **"+"** to add **Overlay Images**. You can position and scale them freely.
5.  Click **Download Image** to save your creation.

## Deployment
This project is static HTML/JS/CSS and can be hosted on GitHub Pages, Vercel, or Netlify.
